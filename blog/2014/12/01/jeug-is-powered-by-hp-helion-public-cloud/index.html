日本 Eucalyptus ユーザ会は HP Helion Public Cloud で動いています

これは Eucalyptus Advent Calendar 2014 1日目の投稿です。

皆さんこんにちは、日本 Eucalyptus ユーザ会です。

当ユーザ会は発足してからこれまで、ML や公式サイトの運営の殆どを前佛氏に頼っておりました。もちろんこれからも氏の協力なくしては運営が成り立たないのですが、例えば公式サイトを稼動させるサーバなどは他者の助力を得ることはできないものか？と常々考えておりました。

そうしたところ、日本 hp の真壁氏より hp Helion Public Cloud で当ユーザ会の公式サイトを稼動させることを提案頂き、当ユーザ会はこれを辞退する理由がないため有り難くこれを受けることにしました。そしてここに当ユーザ会の公式サイトが hp Helion Public Cloud で動いていることを発表致します。

今年の 9/11 に hp 社が Eucalptus 社を買収することを発表し、ある意味ではソフトウェアとしての Eucalyptus は hp 社の庇護のもと発展していくと言えるのではないかと思いますが、日本 Eucalyptus ユーザ会も hp 社の厚意によって発展していきたいと思います。改めて、この提案をして頂いた真壁氏とこの提案を実現して頂いた hp 社には感謝を申し上げます。

最後に、これまで長い間尽力して頂いた前佛氏に深く感謝致します。氏にはこれからも当ユーザ会の発起人および最古のメンバーとして関って頂けると幸いに存じます。

Eucalyptusとは

Eucalyptus は、オープンソース・ソフトウェアによるクラスタリング環境を用いた、可用性のあるユーリティ・コンピューティングのアーキテクチャ、いわゆる “クラウド・コンピューティング” を実用化するためのシステムです。オープンソースなので、誰でも自由に使うことができます。
