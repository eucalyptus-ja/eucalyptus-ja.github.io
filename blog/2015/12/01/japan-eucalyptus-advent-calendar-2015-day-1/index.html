日本 Eucalyptus ユーザ会は HP Helion Public Cloud で動いていました

これは (2枚目) Japan Eucalyptus Advent Calendar 2015 1日目の投稿です。

皆さんこんにちは、日本 Eucalyptus ユーザ会です。

この JEUG のサイトは、旧日本 hp に在籍していた真壁氏(現在は日本マイクロソフトに在籍)のご協力と旧日本 hp さん──現日本ヒューレット・パッカードさんのご厚意により hp Helion Public Cloud で動いておりました。

「ました」と言うのは、皆さんもご存知のとおり、 hp Helion Public Cloud が 2016 年 1 月末にサービスを停止してしまうため、いま皆さんが見ているこのサイトは別のところで動いております。

移行先については後述しますが、これまで当サイトを置かせて頂いた hp Helion Public Cloud および日本ヒューレット・パッカードさんには大変感謝しております。直接お礼の言葉を述べる機会がなく言えませんでしたが、本当に今迄ありがとうございました。今後も JEUG は日本ヒューレット・パッカードさんのご発展と益々のご隆盛をお祈り申し上げます。

さて、前述したとおり本サイトは現在別の場所で稼働しており、その場所とは Microsoft 社の Azure になります。い、いえ、決して真壁氏を追い掛けてストーカーのごとく移行したわけではありません。たまたまの偶然でございます。

今後また別の場所に移行するかもしれませんが、とりあえず暫くは Azure 上で当サイトを動かしてみようと思います。(結構サクサクと動いています)

では、今後とも当ユーザ会および Eucalyptus を宜しくお願い致します。

2015/12/01 日本 Eucalyptus ユーザ会 メンバー全員より
